My Projects

Sudoku Game

Implementation of the classic Sudoku game. It generates new puzzles, allows user to solve and view the solution. For solving, recursive backtracking and Crook's algorithm is used and their time efficiency is compared. GUI is provided using pygame.

A password manager application to store all of your passwords in a sql database in an encrypted format. Developed in Python using sqlite3 module and tkinter for GUI.

Timetable-generator

An exam timetable scheduler that takes in batches and courses as input and creates a time-table using minimum time-slots. Created in JAVA. Graph data structure and graph-coloring algorithm is used.
